Cấu Hình Kết Nối Router

Địa chỉ HTTPS của Cloudflare Tunnel kết nối tới máy chủ Python ở nhà của bạn. Nếu chạy nội bộ, để trống hoặc điền URL backend chạy cục bộ.
IP của Router HG8245H trên mạng cục bộ ở nhà.
Dán trực tiếp Cookie header lấy được từ Tab Network trên Chrome (F12).

Hướng Dẫn Sử Dụng

1
Lấy Cookie thủ công (nếu Đăng nhập tự động lỗi):
Đăng nhập vào router 192.168.1.1 trên máy tính, nhấn F12 -> tab Network -> Chọn một request bất kỳ (ví dụ: `lanuserinfo.asp`) -> tìm mục Request Headers -> Copy toàn bộ dòng Cookie dán vào phần cài đặt.
2
Thiết lập Cloudflare Tunnel (Miễn phí & An toàn):
Tải `cloudflared` về máy tính/Android/Pi chạy backend và chạy lệnh:
cloudflared tunnel run --url http://localhost:5000
Cloudflare sẽ cấp 1 đường dẫn HTTPS miễn phí (ví dụ: https://xxx.trycloudflare.com). Điền link đó vào ô API Server Backend URL trên web Vercel.
3
Đặt biệt danh cho thiết bị:
Ở tab Thiết bị kết nối, bấm vào cột Hành động -> Chọn Đổi tên để lưu biệt danh thiết bị vào database sqlite cục bộ, giúp dễ quản lý.
